WSDL

Available methods:

  • GetFiscalInfo

Get taxpayer fiscal info from database

Parameters:

  • rif: taxpayer RIF

Output: "General code||General message||Specific message." If taxpayer exists in database the specific message will have his fiscal data in the format "RIF;;BUSINESS NAME;;ADDRESS"

  • PostFiscalInfo

Method used to save a customer fiscal data in a database

Parameters:

  • RIF: taxpayer RIF
  • businessName: taxpayer business name
  • address: taxpayer fiscal address

Output: "General code||General message||Specific message."

  • ParseRIFFile

Method used to obtain fiscal data of a taxpayer from the RIF document generated by SENIAT

Parameters:

  • fileEncodedInBase64: base64 encoded string of a PDF RIF file

Ouput: "General code||General message||Specific message." (If given valid RIF file the specific message will show the fiscal data in the format "RIF;;BUSINESS NAME;;ADDRESS")

  • ValidateRIF

Method used to verify if a venezuelan RIF is valid

Parameters:

  • rif: taxpayer RIF.

Output: "General code||General message||Specific message."
